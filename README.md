# FunctionPlus
Minecraft Java 26.2のプラグインです
-機能一覧-
/kotei Mobの固定ON/OFFを切り替えることが出来ます。
/mobkoe Mobの鳴き声ON/OFFを切り替えることが出来ます。
/mobtoumei Mobの透明化ON/OFFを切り替えることが出来ます。
/moblook Mobの輪郭線の表示を切り替えることが出来ます。
/functionpluslog 他プレイヤーのコマンド履歴を見ることが出来ます。

-注意-
対象はプレイヤーが見ている Mob です。
操作できる距離は約10ブロック以内です。
プレイヤーとアーマースタンドは対象外です。
透明化、固定、無音化の状態は Mob に保存されます。
チャンク再読み込み後も、対象 Mob の状態は復元されます。
/moblook を使うには ProtocolLib が必要です。ProtocolLib がない場合、透明化・固定・無音化は使えますが、透明 Mob の輪郭表示は使えません。
固定 Mob は AI が無効になり、移動イベントもキャンセルされます。演出用 Mob や展示用 Mob に向いています。
ビルド設定では Java 25、Paper API 26.2.build.34-alpha を使っています。サーバー環境と Java バージョンの一致に注意が必要です。

FunctionPlus は、サーバー内の Mob を演出・管理しやすくするプラグインです。見ている Mob を透明にしたり、動かないように固定したり、鳴き声を消したりできます。透明にした Mob は通常見えませんが、ProtocolLib が入っていれば /moblook で輪郭だけを表示できます。実行履歴もログとして残るので、誰がどの操作をしたか確認できます。
